Eventos: Focus e Blur

Os eventos focus e blur são eventos de interação do usuário em JavaScript que são acionados quando um elemento HTML recebe ou perde o foco, respectivamente.

O evento focus é acionado quando um elemento HTML é selecionado ou recebe o foco de entrada, geralmente por meio do mouse ou teclado. Isso pode acontecer, por exemplo, quando um usuário clica em um campo de entrada de texto.

Aqui está um exemplo simples de como usar o evento focus em JavaScript:

const inputElement = document.querySelector('input');

            inputElement.addEventListener('focus', function() {
            console.log('O campo de entrada recebeu o foco!');
            });

No exemplo acima, estamos selecionando o primeiro elemento <input> encontrado no documento HTML usando document.querySelector. Em seguida, adicionamos um ouvinte de eventos para o evento focus usando o método addEventListener. Quando o elemento recebe o foco, a função de retorno de chamada será executada e exibirá a mensagem no console.

Por outro lado, o evento blur é acionado quando um elemento HTML perde o foco, geralmente quando o usuário clica fora do elemento ou pressiona a tecla "Tab" para navegar para o próximo elemento.

Aqui está um exemplo simples de como usar o evento blur em JavaScript:

const inputElement = document.querySelector('input');

            inputElement.addEventListener('blur', function() {
            console.log('O campo de entrada perdeu o foco!');
            });

No exemplo acima, o código é semelhante ao exemplo anterior, mas usamos o evento blur em vez do evento focus. Quando o campo de entrada perde o foco, a mensagem será exibida no console.

Esses eventos focus e blur são úteis para realizar ações ou validações quando um elemento HTML ganha ou perde o foco, permitindo que você crie uma experiência interativa e responsiva para seus usuários.

Eventos: Interface do usuário

Aqui está uma lista mais abrangente de eventos JavaScript relacionados à interface do usuário:

  1. Eventos de clique:
    • click: Ocorre quando um elemento é clicado pelo usuário.
    • dblclick: Ocorre quando um elemento é clicado duas vezes rapidamente.
    • contextmenu: Ocorre quando o botão direito do mouse é clicado em um elemento.
  2. Eventos de teclado:
    • keydown: Ocorre quando uma tecla é pressionada.
    • keyup: Ocorre quando uma tecla é liberada.
    • keypress: Ocorre quando uma tecla é pressionada e liberada.
  3. Eventos de formulário:
    • submit: Ocorre quando um formulário é submetido.
    • input: Ocorre quando o valor de um campo de entrada é alterado.
    • change: Ocorre quando o valor de um campo de entrada é alterado e o foco é perdido.
    • focus: Ocorre quando um elemento recebe o foco.
    • blur: Ocorre quando um elemento perde o foco.
  4. Eventos de mouse:
    • mouseenter: Ocorre quando o cursor do mouse entra em um elemento.
    • mouseleave: Ocorre quando o cursor do mouse deixa um elemento.
    • mouseover: Ocorre quando o cursor do mouse passa por cima de um elemento.
    • mouseout: Ocorre quando o cursor do mouse sai de cima de um elemento.
    • mousemove: Ocorre quando o cursor do mouse se move sobre um elemento.
  5. Eventos de carregamento:
    • load: Ocorre quando um recurso, como uma imagem ou um documento, é completamente carregado.
    • DOMContentLoaded: Ocorre quando o DOM (Documento Object Model) da página é completamente carregado.
    • unload: Ocorre quando a página está sendo descarregada (geralmente quando o usuário fecha a janela).

Esses são apenas alguns exemplos de eventos JavaScript relacionados à interface do usuário. Existem muitos outros eventos disponíveis, como eventos de arrastar e soltar (drag and drop), eventos de redimensionamento de janela, eventos de toque (para dispositivos móveis) e muito mais. Você pode consultar a documentação do JavaScript para obter uma lista completa de eventos e suas descrições detalhadas.

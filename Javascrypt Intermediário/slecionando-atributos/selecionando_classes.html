Recuperando Classes de Elementos com JavaScript

Para recuperar as classes de um elemento usando JavaScript, você pode usar os métodos classList ou className. Aqui estão as instruções para cada método:

Usando classList:


// Selecionar o elemento pelo ID
const elemento = document.getElementById('meuElemento');
// Recuperar as classes como um objeto DOMTokenList
const classes = elemento.classList;

// Iterar sobre as classes usando um loop
for (let i = 0; i < classes.length; i++) {
const classe = classes[i];
console.log(classe);
}

Nesse exemplo, primeiro selecionamos o elemento desejado usando getElementById. Em seguida, atribuímos elemento.classList à variável classes, que retorna um objeto DOMTokenList contendo todas as classes do elemento. Podemos então iterar sobre as classes usando um loop e imprimir cada classe no console.

Usando className:


// Selecionar o elemento pelo ID
const elemento = document.getElementById('meuElemento');
// Recuperar as classes como uma string
const classes = elemento.className;

// Dividir a string em um array de classes individuais
const classesArray = classes.split(' ');

// Iterar sobre as classes usando um loop
for (let i = 0; i < classesArray.length; i++) {
const classe = classesArray[i];
console.log(classe);
}

Nesse exemplo, usamos getElementById para selecionar o elemento desejado. Em seguida, atribuímos elemento.className à variável classes, que retorna uma string contendo todas as classes do elemento. Usamos o método split para dividir a string em um array de classes individuais, separadas por espaços. Podemos então iterar sobre o array de classes e imprimir cada classe no console.

Essas são duas maneiras comuns de recuperar as classes de um elemento usando JavaScript. Cada método tem suas próprias peculiaridades e você pode escolher o que melhor se adequa às suas necessidades.
